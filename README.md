GRUB Theme inspired by Cyberpunk 2077.

## Installation

1. Clone the repo

```shell
git clone https://github.com/adnksharp/CyberGRUB-2077
```

2. Go into the folder 

```shell
cd CyberGRUB-2077
```

3. Execute the installation script as admin

```shell
sudo ./install.sh
```

	or 

```shell
sudo $SHELL ./install.sh
```

![](img/out.svg)

3. Test your new theme
